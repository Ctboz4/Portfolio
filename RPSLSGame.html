This lightweight application allows users to enjoy the classic game of 'Rock-Paper-Scissors-Lizard-Spock' with an interactive and straightforward interface.

Highlights

This game allows the user to play against the computer

This application utilizes the Random class to simulate computer decisions

This application keeps up with scores from both the Player and the Computer, as well as the amount of games played

What I learned

  • I gained experience using the Random class to let the computer make its choice
